# Option A: Glass Bowl

Minimal Liquid Glass app icon direction for Mancala.

Import these layers into Icon Composer in numeric order:
1. `01-background.svg`
2. `02-glass-bowl.svg`
3. `03-stones.svg`
4. `04-highlight.svg`

Recommended Icon Composer settings:
- Keep the background as a subtle blue/slate gradient.
- Apply Liquid Glass to the bowl and stones as separate groups.
- Use Combined mode for the bowl group and Individual mode for the stones group.
- Keep specular on; avoid adding extra text or board detail.
